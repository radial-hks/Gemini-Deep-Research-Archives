## 📖 这是一个什么项目？

Gemini 的 **Deep Research（深度研究）** 功能能够针对复杂议题生成结构化、引经据典的高质量报告。然而，这些极具价值的“知识切片”往往散落在个人的聊天记录中。

**GDRA** 致力于将这些共性问题的研究成果汇聚一处，旨在：
1.  **减少重复劳动**：别人研究过的问题，你不需要再跑一次 Prompt。
2.  **Prompt 学习**：通过查看优质报告的原始 Prompt，学习如何更精准地向 AI 提问。
3.  **人类校验**：通过社区的力量，对 AI 生成的内容进行事实核查与补充。

## 🤝 如何贡献 (Contribution)

我们非常欢迎你的加入！无论你是否懂编程，都可以轻松贡献。

### 🟢 方式一：我是普通用户 (推荐 ✨)
**不需要懂 Git，不需要懂代码。** 就像填问卷一样简单。

1. 点击 **[这里提交新研究](https://github.com/radial-hks/Gemini-Deep-Research-Archives/issues/new/choose)** 。
2. 选择 **"🆕 提交 Deep Research 报告"**。
3. 在表单中填入你的 `Prompt` 和 Gemini 生成的 `内容`。
4. 点击提交，我们的志愿者会整理入库！

### 🔵 方式二：我是开发者 (GitHub Veteran)
如果你熟悉 Git 工作流，欢迎直接提交 PR：

1. Fork 本仓库。
2. 复制模板 `templates/research-template.md`。
3. 在 `archives/对应分类/` 下创建新文件，文件名格式：`YYYY-MM-DD-topic-slug.md`。
4. 提交 Pull Request。
